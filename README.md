# Portfolio & Blog Website

A beautiful, fully responsive portfolio and blog website built with HTML, CSS, and JavaScript. This project showcases modern web design practices using semantic HTML5 and clean CSS.

## Features

✨ **Responsive Design**
- Mobile-first approach
- Works seamlessly on all devices (mobile, tablet, desktop)
- Media queries for optimal viewing experience

📄 **Multiple Pages**
- **Home Page** - Hero section, about, featured projects, and latest blog posts
- **Portfolio Page** - Showcase of projects with detailed descriptions
- **Blog Page** - Collection of articles on web development topics

🎨 **Modern Styling**
- Clean, professional design
- CSS Grid and Flexbox layouts
- Smooth transitions and hover effects
- Consistent color scheme and typography

♿ **Accessibility**
- Semantic HTML5 markup
- Proper heading hierarchy
- Accessible navigation
- Good color contrast

⚡ **Performance**
- Lightweight and fast-loading
- Minimal dependencies
- Optimized CSS
- Clean, maintainable code

## How to Use

1. **Clone or download the repository**
   ```bash
   git clone https://github.com/agyangg/ggweb.git
   cd ggweb
   ```

2. **Open in a browser**
   - Simply open `index.html` in your favorite web browser
   - No server or build tools required!

3. **Customize for your needs**
   - Edit the content in HTML files to add your information
   - Replace "Your Name" with your actual name
   - Update project descriptions and links
   - Modify colors and styling in `styles.css`

## Customization Guide

### Changing Your Name
Replace "Your Name" and "YourName" throughout the HTML files with your actual name.

### Updating Contact Information
In `index.html`, update:
- Email address
- LinkedIn profile
- GitHub profile

### Adding/Modifying Projects
Edit the portfolio items in `portfolio.html`:
- Project title
- Description
- Technologies used
- Live site link

### Adding/Modifying Blog Posts
Edit the blog posts in `blog.html`:
- Post title
- Publication date
- Content

### Customizing Colors
Edit the CSS variables and colors in `styles.css`:
- Primary color: `#0066cc`
- Gradient colors: `#667eea` and `#764ba2`
- Background colors: `#f8f9fa`, `#fff`

## Color Scheme

- **Primary Blue**: #0066cc
- **Purple Gradient**: #667eea to #764ba2
- **Light Background**: #f8f9fa
- **Dark Text**: #1a1a1a, #333
- **Footer**: #1a1a1a
