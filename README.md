# Computer Science Teacher Portfolio & Blog

**Gyan Guragain** - Computer Science Teacher at St. Xavier's School, Godavari, Lalitpur, Nepal

## 📚 Overview

This is a comprehensive portfolio and blog website for a Computer Science teacher, built entirely with HTML and CSS. It showcases teaching experience, courses, and educational blog posts.

## ✨ Features

### 🎯 Responsive Design
- Mobile-first approach
- Works seamlessly on all devices
- Media queries for optimal viewing

### 📖 Multiple Pages
- **Home Page** - Introduction, featured courses, blog preview
- **About Page** - Professional background, teaching philosophy, expertise areas
- **Courses Page** - 6 comprehensive courses with detailed descriptions
- **Blog Page** - 4 in-depth articles on computer science topics

### 💻 Course Offerings
1. **Introduction to Python Programming** - Beginner level (12 weeks)
2. **Web Development: HTML, CSS & JavaScript** - Beginner to Intermediate (14 weeks)
3. **Data Structures & Algorithms** - Intermediate (16 weeks)
4. **Object-Oriented Programming (OOP)** - Intermediate (12 weeks)
5. **Database Design & SQL** - Intermediate (10 weeks)
6. **Competitive Programming & Problem Solving** - Advanced (12 weeks)

### 📝 Blog Topics
1. Getting Started with Python Programming
2. Web Development: From HTML to Interactive Sites
3. Understanding Data Structures: Building Blocks of Programming
4. Object-Oriented Programming: Organizing Code with OOP

### 🎨 Modern Styling
- Clean, professional design
- CSS Grid and Flexbox layouts
- Smooth transitions and hover effects
- Consistent color scheme (Blue #0066cc & Purple gradient)
- Accessible design principles

### ♿ Accessibility

## 🚀 How to Use

1. **Clone or download the repository**
   ```bash
   git clone https://github.com/agyangg/ggweb.git
   cd ggweb
   ```

2. **Open in a browser**
   - Simply open `index.html` in your web browser
   - No server or build tools required!
   - Navigate using the menu to explore all pages

3. **Deploy to GitHub Pages**
   ```bash
   # The site is already ready for GitHub Pages
   # Just enable it in repository settings
   ```

## 🎨 Customization Guide

### Update Personal Information
- Replace "Gyan Guragain" with your name throughout HTML files
- Update email addresses: `g.guragain@sxg.edu.np`
- Modify school information as needed
- Update social media links (GitHub, LinkedIn)

### Modify Course Content
- Edit course descriptions in `courses.html`
- Add new courses by copying course-detail block
- Update course duration, level, and modules
- Add specific projects relevant to your teaching

### Add New Blog Posts
- Add new article blocks in `blog.html`
- Follow the existing blog-post structure
- Include meaningful headings and sections
- Add code examples and best practices

### Customize Colors
Edit primary colors in `styles.css`:
- Primary Blue: `#0066cc`
- Purple Gradient: `#667eea` to `#764ba2`
- Light Background: `#f8f9fa`
- Dark Text: `#1a1a1a`, `#333`
